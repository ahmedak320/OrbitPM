Deployment

Get OrbitPM running on your infrastructure in minutes.

Deployment Steps

1

Provision Your Server

Choose a VPS or dedicated server that meets the requirements below. We recommend Ubuntu 22.04 LTS or Debian 12.

2

Install Docker & Docker Compose

OrbitPM ships with Docker Compose configuration. Install Docker and clone the deployment repository to your server.

3

Configure Environment

Set up your environment variables including database connection, encryption keys, and SMTP settings for email notifications.

4

Launch with Docker Compose

Run docker compose up -d to start all services. The app will be available on port 3000 with WebSocket on port 1234.

5

Configure SSL & Domain

Set up Nginx or Traefik as a reverse proxy with SSL certificates. Point your domain to the server and you're live.
